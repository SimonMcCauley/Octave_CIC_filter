Octave script to load a CSV file with MI receiver data in a CSV file and convert to its envelope using CIC filter.
For CSV file, first column is raw time vales, no labels. Sample rate is 524288 sample per second. 
Second column is ADC values, no labels.
Plots show number of bits used by each stage of the CIC filter, input signals and extracted envelope signal.
Sample CSV file inluded for testing
